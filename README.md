# TinDog

A modern, fully responsive landing page for TinDog, a dog matchmaking platform. Developed using HTML5, CSS3, and Bootstrap 4, incorporating a dynamic testimonial carousel, tiered pricing cards, feature showcases, and strategically placed call-to-action elements.

## 🛠️ Tech Stack

- **HTML5**
- **CSS3**
- **Bootstrap 4**
- **Font Awesome 5**
- **Google Fonts** (Montserrat, Ubuntu)

## 📋 Features

- Responsive navigation bar with mobile toggle
- Hero section with app download buttons (iOS & Android)
- Feature highlights with icons
- Interactive testimonial carousel
- Press/media logo showcase
- Three-tier pricing cards (Chihuahua, Labrador, Mastiff)
- Call-to-action download section
- Social media footer

## 🚀 Getting Started

1. Clone the repository
   ```bash
   git clone https://github.com/<your-username>/TinDog.git
   ```
2. Open `index.html` in your browser
